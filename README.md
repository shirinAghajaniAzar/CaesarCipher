# 🔐 Caesar Cipher

This is a beginner Python practice project built to learn the basics of functions, loops, and string manipulation.

The program lets you encrypt and decrypt messages using the classic Caesar cipher — shifting letters in the alphabet by a chosen number.

## What I practiced
- Defining and calling functions
- Working with lists and string indexing
- Using while loops and user input
- Handling edge cases (non-alphabetic characters, large shift numbers)

## Usage

Type 'encode' to encrypt, 'decode' to decrypt

Enter your message and a shift number

The result is displayed instantly

## Requirements
- Python 3
- asciiart module (for the logo)
